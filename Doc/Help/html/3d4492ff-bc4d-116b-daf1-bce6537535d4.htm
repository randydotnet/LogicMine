Filtering support for Web API operations
Classes
  ClassDescription
FilterParser

Parses string filter expressions and produces instances of IFilter which is bound to T.

Example filter expressions are:

MyStringField in ('one', 'two', 'three')

MyStringField in ('one', 'two', 'three') and MyIntField from 1 to 10 and MyDateField lt '2018-06-25'

FilterParserT

Parses string filter expressions and produces instances of IFilter which is bound to T.

Example filter expressions are:

MyStringField in ('one', 'two', 'three')

MyStringField in ('one', 'two', 'three') and MyIntField from 1 to 10 and MyDateField lt '2018-06-25'

FilterTermParser

Parses string terms and produces instances of IFilterTerm.

Example string terms are:

MyStringField eq 'hello'

MyIntField lt 100

MyIntField in (1, 2, 3, 4)

MyDateField from '2017-01-01' to '2017-12-31'

FilterTermParserT

Parses string terms and produces instances of IFilterTerm which are bound to T

Example string terms are:

MyStringField eq 'hello'

MyIntField lt 100

MyIntField in (1, 2, 3, 4)

MyDateField from '2017-01-01' to '2017-12-31'
