DescendToAsync(IGetBasketTId, T)
When processing a "Get" request the cache will be checked to see if the requested object is there. If it is then the cached object and a ReturnNote will be added to the basket and this will cause the shaft to pull the basket back up again, saving on the basket having to decend further down the shaft.
DescendToAsync(IGetCollectionBasketT)
When processing a "Get Collection" request the cache will be checked to see if the requested objects are there. If they are then the cached objects and a ReturnNote will be added to the basket and this will cause the shaft to pull the basket back up again, saving on the basket having to decend further down the shaft.
