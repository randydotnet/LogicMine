CacheLayerTId, T Class

[This is preliminary documentation and is subject to change.]

A layer of stations which provide cache management.

N.B. This layer does not contain a DeleteCollection station and as such is not compatible with mines that contain a DeleteCollection shaft.

When processing a "Get" request the cache will be checked to see if the requested object is there.
